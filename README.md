🎓 Student Management System

A frontend web application to manage student information using HTML, CSS, and JavaScript.

🛠️ Technologies Used

HTML – Structure of the webpage
CSS – Styling and layout
JavaScript – Functionality (Add/Delete students dynamically)

⚙️ Features

➕ Add student details (Name, Roll No, Course)
📋 Display students in a table dynamically
❌ Delete students with a single click
🖥️ Fully client-side (no backend required)
